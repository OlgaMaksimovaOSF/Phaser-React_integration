# Space Shooter Experiment

This repository contains an experimental HTML5 space shooter game that I am building while learning modern web tooling. The goal is to experiment, explore ideas, and have fun while iterating on both the client (Vite/Phaser) and server pieces.

The project currently includes:

- `client/`: Vite-powered frontend with Phaser scenes for the shooter gameplay.
- `server/`: Node.js backend that will support game services as the experiment grows.

This is a personal learning sandbox, so expect rough edges and frequent changes. Feedback and suggestions are welcome!
